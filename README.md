# Compositorio

Sitio estático listo para GitHub Pages.

## Publicación

1. Sube todo el contenido de esta carpeta a la raíz de un repositorio de GitHub.
2. En **Settings → Pages**, selecciona **Deploy from a branch**.
3. Elige la rama `main` y la carpeta `/ (root)`.

No requiere instalación, compilación ni servidor. `index.html` contiene las 12 tarjetas y cada archivo HTML con nombre propio corresponde a una página individual. Las imágenes optimizadas están en `assets/images/` y los estilos compartidos en `styles.css`.
